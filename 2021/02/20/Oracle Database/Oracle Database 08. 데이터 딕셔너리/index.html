Oracle Database 08. 데이터 딕셔너리

Feb 20, 2021 in 개발이론, Database

데이터 딕셔너리

  • 시스템 카탈로그라고 부르기도 하며 사용 가능한 데이터 베이스 및 테이블의 정보를 가지고 있는 시스템 테이블이다.
  • 데이터 딕셔너리는 DBMS만이 추가, 수정, 삭제가 가능하며 사용자는 조회만 가능하다.

테이블 목록 조회하기

select * from tab;

테이블 정보 조회하기

desc 테이블명
