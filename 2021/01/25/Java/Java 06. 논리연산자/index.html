Jan 25, 2021 in 개발이론, Java

논리연산자

and

&&는 좌항과 우항의 값이 모두 참일 때 참이 된다.

or

||는 좌우항중 하나라도 참이라면 참이 된다.

not

!는 부정의 의미로 boolean의 값을 역전시킨다.
